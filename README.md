# Horizon Registration Analysis

## Project Overview
Comprehensive design analysis and test specifications for the Horizon financial affairs management platform registration feature. This project contains user stories, acceptance criteria, JIRA integration, and automated test cases.

## Figma Design

**Design File**: [Horizon - Registration](https://www.figma.com/design/RL8rm7ASpigsMjmqkHxCmm/Horizon?node-id=4-2)  
**Frame**: 4:2 - Registration Page

![Registration Design](https://www.figma.com/design/RL8rm7ASpigsMjmqkHxCmm/Horizon)

### Design Includes:
- Complete registration form with validation
- Benefits showcase section
- Requirements checklist
- Social authentication options
- Terms & Privacy Policy acceptance
- Support contact options

## Test Automation Framework

### Setup
```bash
# Install dependencies
npm install @cucumber/cucumber playwright

# Run all tests
npm test

# Run specific tags
npm test -- --tags @smoke
npm test -- --tags @critical
npm test -- --tags "@validation and @email"

# Run specific feature
npm test -- tests/features/registration.feature
```
