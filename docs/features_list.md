# Features List - Horizon Registration

## Feature: F-001 - User Registration
**Description**: Enable executors and family members to create an account on the Horizon platform to manage deceased person's financial affairs.

**Design References**: Figma Frame 4:2 (Registration page)

**UI Elements**:
- Registration form with input fields
- Benefits showcase section
- Social login options
- Terms and privacy acceptance

**Status**: Identified

---

## Feature: F-002 - Registration Form Validation
**Description**: Validate all user inputs in the registration form to ensure data quality and security.

**Design References**: Figma Frame 4:2

**UI Elements**:

- Required field markers

**Status**: Identified
